Django Socketio


The aim of this project is to test that we can use websockets (well we are using socketio so not only websockets) with django although Django is created with synchronous idea in mind. To test this example there are 3 demos in the playground.

  • Echo: Echo an string to himself or to all the people that are connected to the echo demo
  • Time: Echo the time of the server
  • Chat: A very simple chat (ideal for testing trolling xD)
For this project we have used various technologies:
  • Django 1.4.1
  • Socket.io 0.9.1
  • Gevent-socketio
Also thanks to this other technologies:
  • Gunicorn for the WSHI HTTP server
  • Bootstrap for the templates
  • Jquery-miniColors for the color picker
  • Dotcloud for the demo hosting
Maybe someone also wants to see the code for deploying custom app in dotcloud in this case is with gunicorn + gevent-socketio + django
